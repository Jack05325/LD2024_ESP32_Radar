# LD2420 Radar ESP32

## Descrizione del Progetto

Questo progetto contiene i file di progettazione PCB per un modulo radar basato sul sensore LD2420 integrato con un microcontrollore ESP32. Il sensore LD2420 è un radar a onde millimetriche a 24GHz per il rilevamento di presenza umana, ideale per applicazioni di domotica e automazione.

### Caratteristiche Principali

- **Sensore Radar**: LD2420 a 24GHz per rilevamento preciso di presenza e movimento
- **Microcontrollore**: ESP32 con connettività WiFi e Bluetooth integrata
- **Formato PCB**: Design compatto e ottimizzato per facile integrazione
- **Output**: Dati di rilevamento elaborabili via WiFi/Bluetooth

### Contenuto del Repository

- **File KiCad**: Schema elettrico (.kicad_sch) e layout PCB (.kicad_pcb)
- **BOM**: Bill of Materials nella cartella `/bom`
- **Gerber**: File pronti per la produzione nella cartella `/gerber_to_order`
- **PDF**: Schema elettrico esportato in formato PDF
- **Backup**: Versioni precedenti del progetto in `/LD2420_Radar_ESP32-backups`

### Software Richiesto

- KiCad 7.0 o superiore per aprire e modificare i file di progetto

### Note di Produzione

I file Gerber pronti per l'ordine sono disponibili nella cartella `gerber_to_order`. Consultare il PDF dello schema per i dettagli sul circuito.
